Covid cases were obtained using this API: https://covid19tracking.narrativa.com/ This API obtains the data from different sources including:

  • Ministerio de sanidad
  • Dipartimento della Protezione Civile de Italia
  • Robert Koch Institute de Alemania
  • Sant√© public France
  • Johns Hopkins University

The maps to plot the data were obtained from: https://www.naturalearthdata.com/downloads/10m-cultural-vectors/

Population data was obtained from this API: https://pypi.org/project/countryinfo/

Data was processed and then stored in a json file that can be obtained in the github repository stated above. After that, all maps were created using python (specifically it was created with geopandas which is a variation of matplotlib). Then, videos were also created automatically. All parameters like the cases by 100.000 people were also created by a script.

Finally, this web application is just a user-friendly way of consulting all of this existing data that was created (the application is created with Django and Js).
